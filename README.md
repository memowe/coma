# coma

**A web-based concept map tool**  
[![Build Status](https://travis-ci.org/memowe/coma.svg?branch=master)](https://travis-ci.org/memowe/coma)

## Features

- TGF import/export
- Force-directed graph visualization
- Fast concept map creation
- Graph analysis
